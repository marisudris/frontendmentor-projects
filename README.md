### What I learned

Styling and flex centering bullet points on `ul` elements by using `::marker` seems to be impossible, unless I missed something. Instead I used the old-school `::before` method, and inserting + styling custom bullet points
myself, which *can* feel a bit hacky at times.

### Continued development
